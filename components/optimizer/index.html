Optimizer GSSF Component

This is an extremely basic component, wrapping GMSH for mesh optimization.

Configuration

The second-level GSSF-XML element is as follows:

   <optimizer [ method="(gmsh|netgen):gmsh" ] />

The method corresponds to either the -optimize or -optimize_netgen optimization flag of GMSH.
